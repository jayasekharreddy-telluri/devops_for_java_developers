# **Jenkins Setup on EC2 ‚Äì Commands & Explanation**

##1Ô∏è‚É£ Install Java 17 (required for Jenkins and Maven)

```bash
sudo dnf install java-17-amazon-corretto -y
java -version
```

**Why:**

* Jenkins requires Java to run.
* Verify Java is installed correctly.

---

## 2Ô∏è‚É£ Install Maven (for Java project builds)

```bash
sudo dnf install maven -y
mvn -version
```

**Why:**

* Maven is needed to build Java/Spring Boot projects.
* `mvn -version` ensures Maven is installed and uses the correct Java version.

---

## 3Ô∏è‚É£ Add Jenkins Repository

```bash
wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat-stable/jenkins.repo
```

**Why:**

* Jenkins is not available in default Amazon Linux repos.
* This adds the official Jenkins repository so we can install the latest stable Jenkins package.

---

## 4Ô∏è‚É£ Import Jenkins GPG Key

```bash
rpm --import https://pkg.jenkins.io/redhat-stable/jenkins.io-2023.key
```

**Why:**

* GPG key ensures the Jenkins package is authentic and secure.
* Prevents ‚Äúpackage not trusted‚Äù errors during install.

---

## 5Ô∏è‚É£ Install Jenkins

```bash
dnf install jenkins -y
```

**Why:**

* Installs Jenkins server package from the official repo.

---

## 6Ô∏è‚É£ Start Jenkins Service

```bash
systemctl start jenkins
```

**Why:**

* Starts the Jenkins service so it can run in the background.
* Needed before accessing Jenkins in the browser.

---

## 7Ô∏è‚É£ Check Jenkins Status

```bash
systemctl status jenkins
```

**Why:**

* Confirms Jenkins is **running correctly**.
* Shows active process, memory usage, logs.

---

## 8Ô∏è‚É£ Enable Jenkins on Boot

```bash
systemctl enable jenkins
```

**Why:**

* Ensures Jenkins **starts automatically** when the EC2 instance restarts.

---

## 9Ô∏è‚É£ Get Initial Admin Password

```bash
cat /var/lib/jenkins/secrets/initialAdminPassword
```

**Why:**

* Jenkins requires a one-time **initial admin password** to unlock the web UI.
* Ensures secure first-time setup.

---

## 10Ô∏è‚É£ Access Jenkins UI

* Open in browser:

```
http://<EC2-PUBLIC-IP>:8080
```

**Why:**

* Access the Jenkins dashboard.
* Use the admin password from previous step to unlock.

---

## 11Ô∏è‚É£ Install Suggested Plugins (via UI)

* Click **Install suggested plugins** on the unlock screen.

**Why:**

* Installs essential plugins for CI/CD, Git, Maven, Pipeline, Docker, etc.
* Simplifies setup for beginners and interview purposes.
